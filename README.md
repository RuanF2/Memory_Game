# 🃏 Jogo da Memória

Projeto desenvolvido para praticar fundamentos de HTML5, CSS3 e JavaScript.

## 🚀 Funcionalidades
- **Contador de Tentativas:** Registra quantas vezes o jogador tentou encontrar os pares.
- **Animação de Vitória:** Título interativo que celebra o fim do jogo.
- **Botão Reset:** Efeito Neon moderno com interações ao passar o mouse.
- **Responsividade:** O jogo se adapta perfeitamente a celulares, tablets e desktops.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Variáveis, Flexbox, Animações e Media Queries)
- JavaScript (DOM, Array shuffle, Timer functions)

## 🎨 Design
O projeto conta com um fundo dinâmico de alta resolução e elementos visuais que utilizam efeitos de transparência e brilhos neon.
